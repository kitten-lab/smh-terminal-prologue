# Silo Terminal Prolog

An experimental terminal-based narrative prototype exploring memory, identity, and recursive thinking mechanics.

Built in Python as an early exploration of parser-based interaction, the game explores what it means to create the world from nothing but your own thoughts, even if you can't remember who it was you were.

## Running the prototype

Requirements: 
Python 3.x

Run:
In terminal-
python3 prototype/smh.py

## Status

This project is an early prototype and work in progress.
The goal is explore the mechanics of parser-based adventure games through an internal, rather than external, lens. Using verbs like think and remember, the player will generate the world around them using only their cognition.
